# uni-app-template

基于 `uni-app`、`Vue 3`、`TypeScript`、`Pinia`、`UnoCSS` 和 `uni-helper/unh` 的多端项目模板。

## 特性

- `Vue 3` + `<script setup>` + `TypeScript`
- `uni-app` 多端支持
- `Pinia` 状态管理
- `UnoCSS` 原子化样式
- `Vitest` 测试支持
- `ESLint` 代码检查

## 环境要求

- Node.js 18+
- pnpm 10+

## 安装依赖

```bash
pnpm install
```

## 常用命令

```bash
pnpm dev
pnpm build
pnpm type-check
pnpm lint
pnpm test
```

## 目录结构

```text
src/
  components/    通用组件
  layouts/       布局
  pages/         页面
  static/        静态资源
  stores/        Pinia store
  utils/         工具与请求封装
```
